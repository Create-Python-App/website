# Agent Instructions for website

## Overview

Marketing and documentation site for create-node-app.

## Tech Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS v3
- pnpm 10+

## Key Commands

```bash
# Development
pnpm dev

# Build
pnpm build

# Tests
pnpm test
```

## Code Style

- ESLint flat config
- Prettier for formatting
- Conventional commits
